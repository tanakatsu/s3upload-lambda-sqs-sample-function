# S3Upload-Lambda-SQS-sample (Lambda function)

### Deployment

```
gulp deploy
```

### Run locally

At first, you have to rename .env.sample to .env, and edit it.

Next, install npm modules.

```
node install
```

Then,

```
node CreateThumbsWithMessage_driver.js
```
